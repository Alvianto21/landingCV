# Website Curriculum Vitae (CV) Using laravel

## Description
website for curriculum vitae online using laravel 11 application

### Main Feature
- Export PDF
- Search user CV
- Forgot password
- Dynamic input form with vanila js

## Required
- PHP 8.2
- MYSQL
- Laravel breeze
- barryvdh/laravel-dompdf
